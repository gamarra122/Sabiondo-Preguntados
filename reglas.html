Reglas del Juego

1) Solamente se puede jugar "1" vez o más veces cambiando el nombre de usuario.

2) El premio es para el usuario de mayor puntaje.

3) No se puede retroceder a la pregunta anterior para alterar tu respuesta.

4) No se posee la opción de pistas o saltar la pregunta.

5) Se debe registrar al inicio de cada partida.

6) Posee la opción de abandonar la partida pero pierde el puntaje obtenido.

7) Se debe responder "10" preguntas sin la opción de elegir categoría y sin saber cuántas preguntas se realizarán por cada categoría.

8) Se le otorgará 100 puntos por cada respuesta acertada.

9) Si existe más de un jugador con el mismo puntaje, el que se muestra será el primero en conseguir ese puntaje.

10) Diviértete y buena suerte.
